== Description ==

A simple, secure and effortless way to host giveaways on your WordPress blog or website and download your email list.

**Prerequisites**

* You are a blogger
* Free signup to the http://thinktag.co.uk blogger console

**Features**

* Create your giveaway in 5 mins
* Automated winner selection
* Secure access to console to download participant and winners

[See how it works](http://thinktag.co.uk/static/wordpress.html )

== Installation ==

1. Upload the entire 'mygiveaway' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin using 'Plugins' menu in WordPress.
3. Add the short code to your page or post
